# 프로젝트 로드맵

## 현재 릴리스

현재 공개 버전은 **v15.0.0 - SK서린빌딩 핵심 자산 보유세 세무검토**입니다.

공개 Tax 화면은 주소, PNU, 개별공시지가, 건축물 시가표준액, 신탁구조와 재산세 납세의무자가 공식자료로 연결된 SK서린빌딩 한 건을 표시합니다. General, Assurance와 Methodology 화면은 기존 다회사 기능을 유지하고 Deals와 KRX API는 비활성화합니다.

## v15 완료 범위

- SK리츠·SK서린빌딩·2026년 검토범위 고정
- 토지·건축물·부가세목의 법정 산식 재계산
- 세목별 10원 미만 끝수 처리와 처리 전·후 감사추적
- 기준·5%·10%·사용자 설정 보유세 민감도 분석
- 법정 요건과 자산 사실관계 대조
- 주요 세무쟁점, 추가 요청자료, 메모와 CSV·Excel·HTML 내려받기
- 실제 고지세액 미확보와 미대사 상태의 Fail-closed 표시

## v15.x 안정화 후보

1. 2026년 재산세·지역자원시설세 고지서 및 과세내역서 확보
2. 과세내역서의 분리과세 코드와 실제 끝수 처리 방식 대사
3. 과세기준일 현재 등기부등본·신탁원부로 납세의무자 재검증
4. 토지대장·부속지번 자료로 5.3㎡ 면적 차이 해소
5. 소방분 실제 위험유형 코드와 300% 배율 대사
6. 감면·세부담상한·지방자치단체 조정 반영 검토

## v16 확장 조건

다른 자산이나 리츠로 확대하기 전에 주소·PNU, 기준연도 공식 가액, 소유·신탁구조, 재산세 과세구분과 실제 고지자료가 동일 수준으로 확보되어야 합니다. 동일한 자동 테스트와 과세근거 검토를 통과한 자산만 공개 계산범위에 포함합니다.

## 버전 관리 원칙

- 현재 버전은 `VERSION`과 `config.py`에서 관리합니다.
- 기능, 데이터 계약과 공개 범위 변경은 `CHANGELOG.md`에 기록합니다.
- 미확인 항목은 완료로 표시하지 않고 `docs/v15/COVERAGE_REPORT.md`에 남깁니다.
- main 병합과 운영 배포는 별도 릴리스 검토 후 수행합니다.
