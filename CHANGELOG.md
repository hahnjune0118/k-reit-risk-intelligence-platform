# 변경 이력

## v15.0.0 - SK서린빌딩 핵심 자산 보유세 세무검토

- Tax 화면의 분석범위를 SK리츠·SK서린빌딩·2026년으로 고정
- 공식 과세기초자료와 법정 산식을 연결한 단일 자산 보유세 검토 흐름 구현
- 세목별 `10원 미만 끝수 미계산` 기준과 처리 전·후 감사추적 컬럼 추가
- 끝수 처리 후 기준 재계산액을 `1,250,710,930원`으로 갱신
- 기준·5% 상승·10% 상승·사용자 설정 민감도에 동일 계산 엔진 적용
- 토지 분리과세 판단과 건축물·도시지역분·지방교육세·소방분 계산을 명확히 구분
- 토지분 종합부동산세는 과세대상 제외, 관련 농어촌특별세는 해당 없음으로 정리
- 실제 고지세액 미확보와 고지세액 미대사 상태를 화면·메모·내려받기 자료에 일관되게 표시
- 공개 Tax 화면과 문서를 한국어 세무검토 용어로 정비
- 공식자료가 없으면 계산하지 않는 Fail-closed 통제 유지
