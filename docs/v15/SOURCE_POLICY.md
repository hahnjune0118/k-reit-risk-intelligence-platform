# v15 Source 정책

## 우선순위

1. 실제 재산세·종합부동산세 고지서와 과세내역서
2. 국가법령정보센터, 지방자치단체와 중앙행정기관 공식 자료
3. DART 공시, 리츠 공식 홈페이지·IR·투자보고서
4. 리츠정보시스템과 공식 산업기관 자료
5. 보조 공개자료

검색결과 제목, 블로그와 부동산 중개사이트는 발견 보조용으로만 사용하며 세액 Ground Truth로 사용하지 않습니다.

## Source Lineage 요건

계산 입력값은 최소한 원천 유형, URL, 문서명, 기준일 또는 페이지, 수집시각, 해시와 검토상태를 가져야 합니다. PDF에서 추출한 사실은 관련 페이지를 기록합니다.

## PDF 정책

- 원문 PDF는 `.cache/v15/raw_documents`에 저장하고 Git에 커밋하지 않습니다.
- PyMuPDF로 페이지별 텍스트를 추출합니다.
- 텍스트가 없는 페이지만 OCR을 시도합니다.
- OCR 런타임이 없거나 추출이 실패하면 상태와 오류 유형을 Manifest에 남깁니다.
- 같은 사실이 충돌하면 최신 공식자료를 우선하되 충돌 내역을 삭제하지 않습니다.

## 금지된 대체값

- 투자부동산 장부가액 × 임의 공시가격 비율
- 다른 리츠 또는 Peer의 실효세율
- 회사 전체 Snapshot 보유세 추정
- 확인 불가능한 값의 0 입력
- 공식 근거 없는 주소·PNU·소유지분·시가표준액

## 캐시와 재현성

원문은 로컬 캐시와 SHA-256으로 관리하고, 정규화된 사실·출처 메타데이터만 저장소에 커밋합니다. `--refresh-sources`는 원천을 다시 조회하고 `--offline`은 기존 Snapshot과 캐시만 사용합니다.
