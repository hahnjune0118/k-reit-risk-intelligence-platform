# v15 Tax 사용 가이드

## 화면 범위

Streamlit에서 **Tax: 보유세 분석**을 선택하면 SK리츠 핵심 자산 보유세 세무검토가 열립니다. 부제는 **SK서린빌딩**이며 공개 화면에는 회사·자산·납세의무자 선택기가 없습니다.

## 권장 검토 순서

1. 검토 결론과 네 개의 핵심 지표에서 재계산액과 실제 고지세액 미확보 상태를 구분합니다.
2. 공모부동산투자회사 법적 지위, 신탁구조와 재산세 납세의무자를 확인합니다.
3. 목적사업용 토지의 분리과세 요건과 SK서린빌딩 사실관계를 대조합니다.
4. 주소·PNU·개별공시지가와 건축물 시가표준액의 출처를 확인합니다.
5. 토지·건축물·부가세목의 산식과 세목별 끝수 처리 전·후 금액을 확인합니다.
6. 기준·5%·10%·사용자 설정 민감도와 주요 세무쟁점을 검토합니다.
7. 추가 요청자료 목록과 세무검토 메모를 내려받아 후속 검토에 사용합니다.

## 핵심 표시값

- 끝수 처리 후 보유세 재계산액: `1,250,710,930원`
- 끝수 처리 전 산식상 산출세액: `1,250,710,968.55472원`
- 실제 고지세액: 과세내역서 미확보
- 고지세액 대사 상태: 미대사

끝수 처리 전 금액은 감사추적을 위해 보존한 산식상 금액이고, 화면의 주된 합계는 세목별 끝수 처리 후 금액입니다.

## 결과 해석

- 토지분 종합부동산세는 `과세대상 제외`, 관련 농어촌특별세는 `해당 없음`으로 읽습니다.
- 소방분 300%는 공개된 용도·층수와 법정 요건을 연결한 판단이며 실제 고지 코드와 대사되지 않았습니다.
- 민감도 분석은 공식 가액의 기계적 변동에 따른 결과이며 미래 고지세액 예측이 아닙니다.
- 실제 신고·납부 전에는 과세기준일 현재 법령, 지방자치단체 조례, 과세내역서와 고지서를 다시 확인해야 합니다.

## 내려받기 자료

- 보유세 계산내역 CSV
- 보유세 민감도분석 CSV
- 주요 세무쟁점 CSV
- 추가 요청자료 CSV
- 세무검토팩 Excel
- 보유세 세무검토 Markdown·HTML

모든 파일명과 시트명은 한국어로 제공하고, 끝수 처리 전·후 금액과 실제 고지세액 미확보 상태를 동일하게 유지합니다.
