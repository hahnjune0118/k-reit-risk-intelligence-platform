# v15 Tax 안정화 변경 설명

## 변경 목적

SK리츠·SK서린빌딩·2026년 보유세 검토에 세목별 10원 미만 끝수 처리와 한국어 세법 용어를 반영합니다. 공개 화면은 공식 과세기초자료를 확보한 단일 자산만 표시하고 실제 고지자료가 없는 상태를 명확히 구분합니다.

## 주요 변경

- `Decimal` 기반 세목별 끝수 처리 추가
- 처리 전 산출세액, 단위, 방법, 법적 근거, 처리 후 금액과 차이 보존
- `calculated_tax`와 화면의 주된 합계를 처리 후 금액으로 통일
- 기준 재계산액 `1,250,710,930원`, 5% 상승 `1,313,250,630원`, 10% 상승 `1,375,790,350원`으로 갱신
- 법률판정을 세액보다 먼저 제시하는 20단계 Tax 화면 구성
- 토지 분리과세와 건축물·도시지역분·지방교육세·소방분의 별도 과세를 명시
- 토지분 종합부동산세는 과세대상 제외, 관련 농어촌특별세는 해당 없음으로 표시
- 실제 고지세액 과세내역서 미확보와 미대사 상태 유지
- 메모와 CSV·Excel·HTML 내려받기에 처리 전·후 금액 반영
- Tax 모드 선택 시 사이드바 전체에 적용되던 노란색 CSS 제거

## 검토 한계

재계산액은 공개된 공식 과세기초자료와 확인된 법정 산식을 적용한 결과이며 실제 결정세액·확정세액·납부세액이 아닙니다. 실제 분리과세 코드, 끝수 처리 방식, 감면, 세부담상한, 지방자치단체 조정과 소방분 위험유형 코드는 과세내역서로 대사해야 합니다.

## 릴리스 제한

이 변경은 feature branch의 안정화 검토용입니다. main 병합, 운영 배포와 태그 생성은 별도 승인 전까지 수행하지 않습니다.
