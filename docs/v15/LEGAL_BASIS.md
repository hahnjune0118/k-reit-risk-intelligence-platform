# v15 법령 근거

확인일: **2026-07-16**. 아래 법령은 국가법령정보센터의 현행 조문을 기준으로 확인했습니다. 실제 신고·납부 또는 법률의견 작성 전에는 과세기준일 현재 시행 법령, 지방자치단체 조례와 과세내역서를 다시 확인해야 합니다.

| 검토 영역 | 현행 법령·조문 | 엔진 적용과 한계 |
|---|---|---|
| 끝수 처리 | [지방회계법 제55조](https://www.law.go.kr/법령/지방회계법/제55조) | 10원 미만 끝수를 계산하지 않을 수 있는 선택 규정으로 검토 |
| 끝수 처리 예외 | [지방회계법 시행령 제67조](https://www.law.go.kr/법령/지방회계법시행령/제67조) | 분할 징수·수납·지급의 10원 미만 금액 또는 끝수를 최초 금액에 합산하는 예외를 별도 검토 |
| 지방세 공통원칙 | [지방세기본법](https://www.law.go.kr/법령/지방세기본법) | 과세관계와 불복·징수 등 공통 법체계 확인; 본 엔진의 개별 세목 산식을 대체하지 않음 |
| 재산세 납세의무자 | [지방세법 제107조](https://www.law.go.kr/법령/지방세법/제107조) | 과세기준일 현재 사실상 소유자와 신탁재산의 위탁자 판정 |
| 재산세 과세표준 | [지방세법 제110조](https://www.law.go.kr/법령/지방세법/제110조) | 토지·건축물 시가표준액과 공정시장가액비율 연결 |
| 재산세 세율 | [지방세법 제111조](https://www.law.go.kr/법령/지방세법/제111조) | 분리과세 토지 0.2%, 일반 건축물 0.25% 적용 |
| 재산세 도시지역분 | [지방세법 제112조](https://www.law.go.kr/법령/지방세법/제112조) | 적용대상 과세표준에 0.14% 적용 |
| 소방분 지역자원시설세 | [지방세법 제146조](https://www.law.go.kr/법령/지방세법/제146조) | 누진세율과 대형 화재위험 건축물 300% 배율 적용 |
| 지방교육세 | [지방세법 제151조](https://www.law.go.kr/법령/지방세법/제151조) | 도시지역분을 제외한 재산세 본세의 20% 적용 |
| 공모리츠 목적사업용 토지 | [지방세법 시행령 제102조](https://www.law.go.kr/LSW/lsSideInfoP.do?docCls=jo&joBrNo=00&joNo=0102&lsiSeq=286395&urlMode=lsScJoRltInfoR) | 공모부동산투자회사의 목적사업용 토지 분리과세 요건을 사실관계와 개별 대조 |
| 공정시장가액비율 | [지방세법 시행령 제109조](https://www.law.go.kr/법령/지방세법시행령/제109조) | 토지·건축물 70% 적용 |
| 대형 화재위험 건축물 | [지방세법 시행령 제138조](https://www.law.go.kr/법령/지방세법시행령/제138조) | 주거용이 아닌 11층 이상 요건과 공개된 용도·층수 연결 |
| 토지분 종합부동산세 범위 | [종합부동산세법 제11조](https://www.law.go.kr/법령/종합부동산세법/제11조) | 종합합산·별도합산 토지만 과세대상으로 규정하므로 분리과세 토지는 제외 |
| 종합부동산세분 농어촌특별세 | [농어촌특별세법 제5조](https://www.law.go.kr/법령/농어촌특별세법/제5조) | 납부할 종합부동산세액이 있을 때 20% 적용; 선행 세목이 과세대상 제외이면 해당 없음 |

## 법률 검토 결론

1. 공개자료상 공모부동산투자회사 요건, 목적사업용 부동산 보유와 과세기준일 전후 보유 연속성을 연결하면 SK서린빌딩 **토지**는 분리과세 적용요건을 충족하는 것으로 분석됩니다.
2. 분리과세 판단은 토지에만 적용하며 건축물분 재산세, 재산세 도시지역분, 지방교육세와 소방분 지역자원시설세는 별도로 계산합니다.
3. 분리과세 토지는 토지분 종합부동산세 과세대상에서 제외되고, 그에 따른 농어촌특별세는 해당 없음으로 처리합니다.
4. 업무시설·지상 36층이라는 공개자료에 따라 대형 화재위험 건축물 300% 배율을 적용했으나 실제 고지서의 위험유형 코드는 확인하지 못했습니다.
5. 지방회계법 제55조의 끝수 처리 기준은 본 분석에서 선택하여 적용한 기준입니다. 모든 경우 강제되는 확정 규칙이나 실제 고지세액 검증 결과로 표현하지 않습니다.

- 실제 재산세·지역자원시설세 고지세액
