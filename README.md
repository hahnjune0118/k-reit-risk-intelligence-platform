# K-REIT Risk Intelligence Platform

상장리츠의 공시자료, 거시경제 지표, 자산별 정보와 공시가격 데이터를 연결하여 Assurance 및 Tax 관점의 초기 위험 검토를 지원하는 Streamlit 기반 공개 프로젝트입니다.

**현재 버전: v15.0.0 - SK서린빌딩 핵심 자산 보유세 세무검토**

## 프로젝트 개요

DART, ECOS, 리츠 공시와 공시가격 자료는 유용하지만 서로 분리되어 있습니다. 이 프로젝트는 공개자료 수집, 지표 비교, 위험 신호 식별, 계산근거 추적과 추가 요청자료 작성을 하나의 검토 흐름으로 구조화합니다.

현재 공개 화면은 다음 네 모드로 구성됩니다.

1. **일반 정보 및 시나리오**: 상장리츠의 재무·자산 정보와 거시경제 시나리오 요약
2. **Assurance: 감사위험 분석**: 자산 우선순위, RMM(중요왜곡표시위험), KAM(핵심감사사항)과 감사절차 검토
3. **Tax: 보유세 분석**: SK서린빌딩의 토지·건축물 관련 보유세 세무검토
4. **분석 방법론 및 데이터 출처**: 지표 정의, 자료 계보, 보안, 한계와 면책사항

Deals 모드와 KRX 기반 시장가치 분석은 공개 런타임에서 비활성화되어 있습니다.

## v15 Tax 검토 범위

Tax 모듈은 SK리츠의 핵심 자산인 SK서린빌딩을 대상으로 주소, 필지고유번호(PNU), 개별공시지가, 건축물 시가표준액, 신탁구조와 재산세 납세의무자를 연결합니다. 범용 데이터 구조는 유지하지만 공개 화면은 공식 과세기초자료가 확보된 단일 자산만 표시합니다.

| 구분 | 공개 검토 결과 |
|---|---|
| 분석대상 | SK리츠·SK서린빌딩·2026년 |
| 끝수 처리 전 산식상 산출세액 | `1,250,710,968.55472원` |
| 끝수 처리 후 보유세 재계산액 | `1,250,710,930원` |
| 실제 고지세액 | 과세내역서 미확보 |
| 고지세액 대사 상태 | 미대사 |

현행 법령과 공개된 사실관계를 연결하면 SK서린빌딩 토지는 공모부동산투자회사의 목적사업용 토지로서 분리과세 적용요건을 충족하는 것으로 분석됩니다. 이 판단은 **토지**에 한정되며, 건축물분 재산세, 재산세 도시지역분, 지방교육세와 소방분 지역자원시설세는 각각 별도로 계산합니다. 분리과세 토지는 토지분 종합부동산세 과세대상에서 제외되는 것으로 처리하고, 그에 따른 농어촌특별세는 해당 없음으로 표시합니다.

## 끝수 처리와 감사추적

각 세목은 세율과 배율을 적용한 산출세액을 먼저 보존한 뒤, 지방회계법 제55조에 따른 `10원 미만 끝수 미계산` 기준을 적용합니다. 전체 합계를 한 번만 처리하지 않고 다음 7개 세목마다 처리한 후 합산합니다.

- 토지분 재산세, 토지분 재산세 도시지역분, 토지분 지방교육세
- 건축물분 재산세, 건축물분 재산세 도시지역분, 건축물분 지방교육세
- 소방분 지역자원시설세

CSV에는 끝수 처리 전 금액, 단위, 방법, 법적 근거, 처리 후 금액과 차이를 함께 저장합니다. 시가표준액, 과세표준, 세율, 배율과 과세대상 제외·해당 없음 행에는 끝수 처리를 적용하지 않습니다. 지방회계법 제55조는 선택 규정이므로 실제 과세관청의 처리 방식과 시행령 제67조의 예외 여부는 과세내역서로 추가 확인해야 합니다.

## 보유세 민감도 분석

| 민감도 분석 | 토지·건축물 변동률 | 끝수 처리 전 합계 | 끝수 처리 후 합계 |
|---|---:|---:|---:|
| 기준 | 0% | 1,250,710,968.55472원 | 1,250,710,930원 |
| 공시가격·시가표준액 5% 상승 | +5% | 1,313,250,671.982456원 | 1,313,250,630원 |
| 공시가격·시가표준액 10% 상승 | +10% | 1,375,790,375.410192원 | 1,375,790,350원 |

민감도 분석은 미래 결정세액이나 과세관청의 고지세액을 예측하지 않습니다. 개별공시지가와 건축물 시가표준액만 기계적으로 바꾸고 법적 분류, 세율, 공정시장가액비율, 소유지분과 필지면적은 고정합니다.

## 계산 및 통제 구조

```text
공식 과세기초자료
  -> 자산·필지·건축물·납세의무자 등록부
  -> 재산세 과세구분
  -> 세법 규칙표
  -> 세목별 산출세액
  -> 세목별 10원 미만 끝수 처리
  -> 검증·고지세액 대사
  -> 민감도·세무쟁점·추가 요청자료
  -> 세무검토 메모와 내려받기 자료
```

공식 근거가 부족한 값은 장부가액, Peer 비율이나 임의의 0으로 대체하지 않습니다. 실제 고지서 확인 전에는 `verified_notice` 상태를 생성하지 않습니다.

## 데이터 출처와 보안

- 리츠정보시스템, 리츠 공식 홈페이지·IR·PDF와 DART 공시
- ECOS 거시경제 지표와 V-World 등 공시가격 관련 공식자료
- 국가법령정보센터의 지방회계·지방세·종합부동산세 관련 법령
- `data/v15/*.csv` 정규화 Snapshot, 자료 계보와 검증 상태

공개 배포 버전은 Streamlit Secrets 또는 환경변수로 서버 측 인증정보를 관리합니다. API Key는 GitHub, 화면, 로그, 디버그 출력과 내려받기 파일에 표시하지 않습니다. 실시간 연결이 제한되면 검증된 Snapshot을 사용하되, 공식 과세기초자료가 없는 Tax 항목은 예시값으로 채우지 않고 계산을 차단합니다.
